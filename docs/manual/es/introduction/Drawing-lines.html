Dibujando líneas

Supongamos que quieres dibujar una línea o un círculo, no un wireframe [page:Mesh]. Primero, necesitamos configurar el [page:WebGLRenderer renderizador], [page:Scene escena] y cámara (ver página Creando una escena).

Este es el código que utilizaremos:

const renderer = new THREE.WebGLRenderer(); renderer.setSize( window.innerWidth, window.innerHeight ); document.body.appendChild( renderer.domElement ); const camera = new THREE.PerspectiveCamera( 45, window.innerWidth / window.innerHeight, 1, 500 ); camera.position.set( 0, 0, 100 ); camera.lookAt( 0, 0, 0 ); const scene = new THREE.Scene();

Lo siguiente que haremos será definir un material. Para las líneas debemos utilizar [page:LineBasicMaterial] o [page:LineDashedMaterial].

//create a blue LineBasicMaterial const material = new THREE.LineBasicMaterial( { color: 0x0000ff } );

Después del material necesitaremos una geometría con algunos vértices:

const points = []; points.push( new THREE.Vector3( - 10, 0, 0 ) ); points.push( new THREE.Vector3( 0, 10, 0 ) ); points.push( new THREE.Vector3( 10, 0, 0 ) ); const geometry = new THREE.BufferGeometry().setFromPoints( points );

Ten en cuenta que las líneas se dibujan entre cada par de vértices consecutivos, pero no entre el primero y el último (la línea no es cerrada).

Ahora que tenemos puntos para dos líneas y un material, podemos unirlos para crear una línea.

const line = new THREE.Line( geometry, material );

Solo queda añadirlo a la escena y llamar a [page:WebGLRenderer.render render].

scene.add( line ); renderer.render( scene, camera );

Ahora deberías ver una fecha apuntando hacia arriba, formada por dos líneas azules.
