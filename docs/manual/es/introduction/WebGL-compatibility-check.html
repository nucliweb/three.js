Comprobando la compatibilidad con WebGL

Aunque esto es cada vez menos problemático, es posible que algunos dispositivos o navegadores no soporten WebGL. El siguiente método permite comprobar si está soportado y mostrar un mensaje al usuario si no lo está.

Incluye [link:https://github.com/mrdoob/three.js/blob/master/examples/jsm/WebGL.js] en tu JavaScript y ejecuta lo siguiente antes de intentar renderizar algo.

if ( WEBGL.isWebGLAvailable() ) { // Initiate function or other initializations here animate(); } else { const warning = WEBGL.getWebGLErrorMessage(); document.getElementById( 'container' ).appendChild( warning ); }
